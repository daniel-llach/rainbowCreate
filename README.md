# rainbowCreate
A jquery component that allows you to create rainbows textures in to divs.

## Use

```javascript
$('#example').rainbowCreate({
  direction: "vertical",
  reverse: false,
  linesWidth: 45,
  colorRank: [15,90], // 0=red, 60=yellow, 120=green, 180=light blue, 240=blue, 300=pink
  saturation: 80,
  light: 50,
  alpha: 1
});
```
